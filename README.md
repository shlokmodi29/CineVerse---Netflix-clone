# CINEVERSE

CINEVERSE is a modern **Netflix-inspired streaming platform** built using **HTML, CSS, JavaScript, and Node.js**.  
It provides a sleek and responsive UI for browsing movies with smooth interactions and an immersive user experience.

---

## ✨ Features

- 🎥 Modern Netflix-inspired UI
- 📱 Fully Responsive Design
- ⚡ Smooth Navigation & Fast Loading
- 🎞 Interactive Movie Sections
- 🎨 Clean and Minimal Design
- 🚀 Backend support with Node.js

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Version Control:** Git & GitHub

---

## 📂 Project Structure

```bash
CINEVERSE/
│
├── assets/        # Images, posters, media files
├── app.js         # Frontend logic
├── index.css      # Styling
├── index.html     # Main UI
└── server.js      # Backend server
```

---

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/shlokmodi29/CineVerse-Netflix---Clone-.git
```

2. Open project folder

```bash
cd CineVerse-Netflix---Clone-
```

3. Run the project

Open `index.html` in browser  
OR start backend server:

```bash
node server.js
```
